# Exercise 2-1

Create an application with the following features:

1. Implement steps from "Create an Android project", "Run your app", and "Build a simple user interface" in "Build your first app" (see link below).
2. Add 5 **different** type of UI components to your app and arrange them in a nice way on the main activity.
3. Explain briefly what is the purpose of the UI components you selected as a comment in xml.
4. Optional. Can you run the application on your phone?

**NB!** All widgets are dummy without any code added.
